შერჩევის დალაგება არის მარტივი და ინტუიციური დახარისხების ალგორითმი, რომელიც მოქმედებს დაუხარისხებელი სიიდან მინიმალური ელემენტის განმეორებით მოძიებით და სიის დასაწყისში მოთავსებით.

შერჩევის დალაგებას აქვს O(n^2) დროის სირთულე, სადაც n არის ელემენტების რაოდენობა სიაში. მიუხედავად იმისა, რომ ეს არ არის დახარისხების ყველაზე ეფექტური ალგორითმი, ის მარტივია და გამოსადეგია მცირე სიებისთვის ან საგანმანათლებლო მიზნებისთვის, რათა გაეცნონ დახარისხების ალგორითმებს.



def selection_sort(array):
    for i in range(len(array)):
        min_val = array[i]

        # i+1 იმიტომ რომ ჩვენ ავირჩიეთ პირველი ელემენტი როგორც ყველაზე პატარა ასერომ
        # ვიწყებთ ამის წინა ელემენტიდან

        for j in range(i+1, len(array)):
            if array[j] > min_val:
                min_val = array[j]

        min_indx = array.index(min_val, i)
        if array[i] != min_val: # თუ ორი ერთნაირი ელემენტი არ არის
            array[i], array[min_indx] = array[min_indx], array[i]

    return array

მოკლედ პირველ რიგში namespace ვიყენებთ იმ ფაილში სადაც გვაქვს კლასები რომლის იმპორტიც გვინდა

ანუ აქ შევქმენი namespace config; იმ ფაილისთვის საიდანაც მინდა კლსების იმპორტი 

namespace config;

class Person {
    public $name;

    public function __construct($name)
    {
        $this->name = $name;
    }
}

და ამას ვიყენებთ ასე

ფაილის მისამართი
require_once "./config.php";

და კლასის წინ ვიყენებთ namespace რათა არ აგვერიოს სხვა კლასში
echo var_dump(new config\Person('Giorgi'));

ესეიგი თუ გვინდა რო ესე რექუაირები არ ვწეროთ მაშინ შეგვიძლია გამოვიყენოთ composer autoloader

1) ვიწერთ composer

2) გადავდივართ იმ ფაილში სადაც არის ჩვენი პროექტი და ვწერთ composer init

3) მერე დაგენერირდება ფაილი composer.json და ამდროს ტერმინალში უბრალოდ ვწერთ composer install

4) ამის შემდეგ დაგენერირდება vendor ფაილი რომელიც შეგვიძლია გიტ იგნორში მოვათავსოთ

ამდროს ჩვენ გადავდივართ და ვამატებთ ამ ხაზს რექუირეს დაბლა
"autoload": {
    "psr-4": {
      "classes\\": "./classes"
    }
  }

აქ ეხლა "classes\\" ეს არის namespace რომელიც ეხმაურება ამ ფოლდერს "./classes"

5) შემდეგ ვუშვებთ composer dump-autoload და მორჩა ყველაფერი მზად არის

6) ეხლა გვჭირდება რომ require გავაკეთოთ

თუ ფაილი მთავარ ფოლდერშია და სხვაგან არარი შეტანილი ვწერთ ამას
ამას ვწერთ იმ ფოლდერში საიდანაც ვუშვებთ კოდს
require __DIR__.'/vendor/autoload.php';

თუ ფაილი არის რაიმე ფოლდერში ვაკეთებთ ამას
require __DIR__.'/../vendor/autoload.php';

და ეხლა კიდე შეგვიძლია გამოვიყენოთ ის namespace და წამოვიღოთ ჩვენი კლასი

use classes as cl; ასე
